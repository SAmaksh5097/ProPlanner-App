<h1>🚀 ProPlanner: AI-Powered Project Roadmap</h1>
<br/>
ProPlanner is a high-performance MERN stack application designed to transform project ideas into actionable, step-by-step roadmaps using AI.

<h2>✨Features</h2>
AI Roadmap Generation: Instantly creates structured tasks based on project descriptions.

Dynamic UI: Enhanced with Framer Motion for staggered animations and interactive "Living" backgrounds.

Secure Auth: Powered by Clerk for seamless user management.

<h2>⚙️Tech Stack</h2>

Backend: Node.js, Express.js
Database: MongoDB
Auth: Clerk
AI Engine: Google Gemini API
